# Igreja Presbiteriana em Palmeiras

Um portal web para a Igreja Presbiteriana em Palmeiras-BA, com informações institucionais, conteúdos bíblicos, músicas, galeria de fotos, documentos e assistente virtual.

## ✨ Descrição Resumida

Este projeto oferece um site completo para a comunidade da IPB Palmeiras, facilitando o acesso a conteúdos cristãos, agenda de cultos, downloads, pedidos de oração, histórico da igreja e muito mais. Público-alvo: membros, visitantes e interessados na igreja.

## 🎬 Demonstração

Acesse a versão online: [https://ipbpalmeirasba.onrender.com](https://ipbpalmeirasba.onrender.com)

## 🚀 Funcionalidades

- 📖 Versículo diário automático
- 🎵 Player de músicas cristãs (YouTube)
- 🖼️ Galeria de fotos dinâmica
- 📄 Download de documentos institucionais
- 🗓️ Agenda de eventos e cultos
- 🤖 Chatbot assistente virtual (Sarcinha)
- 🌗 Alternância de temas (claro, escuro, verde)
- 🌍 Suporte a múltiplos idiomas (tradução)
- 📊 Contador de visitantes
- 📚 Leitura bíblica online (API)

## 🛠️ Tecnologias Utilizadas

- HTML5, CSS3 (TailwindCSS), JavaScript (ES6)
- Node.js (backend)
- APIs externas: Bible API, YouTube Embed, Google Calendar, Leaflet Maps
- Font Awesome, Google Fonts

## ⚙️ Instalação / Como Rodar

Pré-requisitos: Node.js, npm

```bash
git clone https://github.com/usuario/ipbpalmeirasba.git
cd ipbpalmeirasba
npm install
npm start
```

Acesse `http://localhost:3000` no navegador.
